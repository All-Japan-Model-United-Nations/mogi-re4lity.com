# 事務局へようこそ

このサイトは、模擬国連全日本大会（AJMUN）事務局の活動を紹介し、そのノウハウを共有するためのプラットフォームです。
2023年末の35thAJMUN、2024年末の36thAJMUNの運営メンバーによって構築・運営されています。

## このサイトの目的

### 1. 事務局の魅力を伝える
AJMUN事務局がどのような組織で、どのような想いで活動しているのかを紹介します。
[役職紹介](./roles/intro)のページでは、各ポジションの具体的な業務内容ややりがいをご覧いただけます。
これから事務局に参加したいと考えている方や、模擬国連の運営に興味がある方に、私たちの活動を知ってもらうことを目指しています。

### 2. ノウハウのオープンソース化
過去の大会運営で蓄積されたマニュアルや資料を[資料庫](./soumu_docs/overall)として公開しています。
失敗談や成功体験も含め、ありのままの情報を共有することで、これからのAJMUN、そして模擬国連コミュニティ全体の発展に寄与したいと考えています。

### 3. 新しい技術への挑戦
このサイト自体が、静的サイトジェネレーター（Docusaurus）を用いた新しい運営スタイルの提案でもあります。
サーバーレスで低コスト、かつカスタマイズ性の高い情報発信の形を模索しています。

## サイトの構成

- **[事務局について](./about_us/about-mogi)**: 模擬国連やAJMUNの概要について説明します。
- **[役職・仕事紹介](./roles/intro)**: 事務総長から各局まで、事務局の多様な役割を紹介します。
- **[資料庫](./soumu_docs/overall)**: 実際の運営で使用されたマニュアルや規定類をアーカイブしています。

## GitHub 最新の更新情報

GitHubの直近の5件ぶんの更新情報を表示します。

<div id="commits">
    <LatestCommits />
</div>

export function LatestCommits() {
    const [commits, setCommits] = useState([]);
    const repoOwner = "All-Japan-Model-United-Nations";
    const repoName = "mogi-re4lity.com";
    const apiUrl = `https://api.github.com/repos/${repoOwner}/${repoName}/commits`;

    useEffect(() => {
        async function fetchCommits() {
            try {
                const response = await fetch(apiUrl);
                if (!response.ok) {
                    throw new Error(`GitHub API エラー: ${response.status}`);
                }
                const commitsData = await response.json();
                setCommits(commitsData.slice(0, 5)); // 最新5件を取得
            } catch (error) {
                console.error("API エラー:", error);
            }
        }
        fetchCommits();
    }, []);

    return (
        <div>
            {commits.length > 0 ? (
                commits.map((commit, index) => (
                    <div key={index} style={{ padding: "10px", border: "1px solid #ddd", borderRadius: "5px", marginBottom: "10px" }}>
                        <p><strong>メッセージ:</strong> {commit.commit.message}</p>
                        <p><strong>担当者:</strong> {commit.commit.author.name}</p>
                        <p><strong>日時:</strong> {new Date(commit.commit.author.date).toLocaleString()}</p>
                        <p><a href={commit.html_url} target="_blank" rel="noopener noreferrer">詳細を見る</a></p>
                    </div>
                ))
            ) : (
                <p>最新のコミットを取得中...</p>
            )}
        </div>
    );
}
