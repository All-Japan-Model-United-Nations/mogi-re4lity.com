# 会議監督 (Director)

会議監督は、模擬国連の「質」を決定づける責任者です。
議題の設定からBGの執筆、当日の会議進行まで、アカデミックな側面を統括します。

## 主な役割

- **議題選定**: 大会で扱うテーマを決定し、シミュレーションの枠組みを設計します。
- **BG作成**: 参加者のための背景資料（Background Guide）を執筆・監修します。
- **フロント官統括**: 議長や秘書官などのフロントメンバーを指導・育成します。
- **会議進行**: 大会当日のセッションを円滑に進め、議論を活性化させます。

## やりがい

- 自分の興味ある国際問題を深く掘り下げ、参加者に「学び」を提供できます。
- ロジカルシンキングやファシリテーションスキルが圧倒的に向上します。
- 白熱した議論が生まれ、決議が採択された瞬間の感動はひとしおです。

> 「会議の結果は、あなたの設計次第。最高の舞台を用意して、参加者のポテンシャルを引き出しましょう。」
