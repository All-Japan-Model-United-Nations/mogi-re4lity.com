# 渉外 (External Affairs)

渉外は、AJMUNの外交官です。
企業や他団体とのパートナーシップを構築し、大会の価値を高めます。

## 主な役割

- **協賛獲得**: 企業にアポイントを取り、協賛金や物品提供を依頼します。
- **連携交渉**: 他の模擬国連団体や国際協力団体との連携を模索します。
- **ゲスト招聘**: 開会式の講演者や、勉強会の講師などを招待します。
- **ネットワーキング**: 社会人やOBOGとの繋がりを作り、組織の資産とします。

## やりがい

- 社会人と対等に交渉する経験ができ、ビジネスマナーやプレゼン力が身につきます。
- 自分の行動力次第で、大会に大きなリソースをもたらすことができます。
- 多様な大人との出会いを通じて、視野が広がります。

> 「学生という枠を超えて、社会と繋がりましょう。あなたの熱意が、支援を引き出します。」
