# 財務 (Finance)

財務は、AJMUNの金庫番です。
限られたリソースを最大限に活用し、大会の持続可能な運営を実現します。

## 主な役割

- **予算策定**: 各部署の要望を聞き取り、全体の予算計画を立案します。
- **資金管理**: 参加費の徴収や経費の精算、銀行口座の管理を行います。
- **助成金申請**: 大学や自治体、財団などの助成金に応募し、資金を獲得します。
- **決算報告**: 大会終了後に収支報告書を作成し、透明性を担保します。

## やりがい

- お金の流れを通じて、組織全体の動きを把握できます。
- 交渉力や計数感覚、リスクマネジメント能力が身につきます。
- 健全な財政基盤を築くことで、他のメンバーが安心して活動できる環境を作ります。

> 「お金の管理は、信頼の管理です。正確かつ戦略的な財務運営で、大会の可能性を広げましょう。」
