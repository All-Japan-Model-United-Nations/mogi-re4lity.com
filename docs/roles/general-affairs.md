# 総務 (General Affairs)

総務は、AJMUN運営の要であり、参加者と事務局を繋ぐ架け橋です。
緻密な事務処理とホスピタリティあふれる対応で、大会の成功を下支えします。

## 主な役割

- **参加者管理**: 応募フォームの作成・管理、参加者名簿の作成などを行います。
- **問い合わせ対応**: 参加者からのメールやSNSでの質問に迅速かつ丁寧に対応します。
- **資料作成**: 大会要綱や募集要項、各種マニュアルなどの文書を作成します。
- **ロジスティクス**: 会場の予約や備品の手配、当日の受付運営などを担当します。

## やりがい

- 参加者から直接感謝の言葉をもらえる機会が多く、モチベーションに繋がります。
- マルチタスク能力や、正確な事務処理能力が身につきます。
- 組織の円滑な運営に貢献しているという実感が得られます。

> 「地味な仕事に見えるかもしれません。しかし、総務がいなければ大会は成立しません。あなたこそが、縁の下の力持ちです。」
