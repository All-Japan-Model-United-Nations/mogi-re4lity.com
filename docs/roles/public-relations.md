# 広報 (Public Relations)

広報は、AJMUNのクリエイティブ・ディレクターです。
デザインと言葉の力で、大会のブランディングと集客を担います。

## 主な役割

- **SNS運用**: X (旧Twitter) やInstagramで情報を発信し、ファンを増やします。
- **ウェブサイト管理**: この公式サイトの更新や、新しいコンテンツの企画を行います。
- **クリエイティブ制作**: パンフレット、ポスター、PVなどの制作物をデザインします。
- **ブランディング**: ロゴやカラーなど、大会の統一されたイメージを作り上げます。

## やりがい

- 自分の作ったコンテンツが多くの人の目に触れ、反応が得られます。
- デザインスキルやライティングスキル、マーケティングセンスが磨かれます。
- 大会の「顔」を作り上げる面白さがあります。

> 「あなたのセンスが、参加者の心を動かします。AJMUNの魅力を、世界に届けましょう。」
