# 事務総長 (Secretary General)

事務総長は、AJMUN事務局の顔であり、組織全体のリーダーです。
ビジョンの策定からメンバーのモチベーション管理まで、幅広い責任を担います。

## 主な役割

- **ビジョン策定**: 大会のコンセプトや目指すべき方向性を決定します。
- **組織マネジメント**: 各役職の進捗管理や、メンバー間の連携を促進します。
- **最終意思決定**: 重要な局面での判断や、トラブル時の対応を行います。
- **対外代表**: 関係各所への挨拶や、メディア対応など、組織を代表して行動します。

## やりがい

- 自分の理想とする大会を一から作り上げる達成感があります。
- 多様なメンバーをまとめ上げるリーダーシップが身につきます。
- 組織運営の難しさと面白さを肌で感じることができます。

> 「事務総長は孤独な仕事ではありません。信頼できる仲間と共に、最高の景色を見に行きましょう。」
