# 大会要綱
このページでは大会要綱について解説します。  
35th-36th共通です。

## 大会要綱とは

会議2-3週間ほど前に大会参加者に向けて配布する「しおり」のようなもの。
宿泊に関する諸連絡や持ち物、諸注意などが記載されている。

## 含める情報

- 大会事務総長挨拶
- 大会コンセプト
- 会議詳細

- 大会準備にあたって
  - 当日の持ち物・服装
  - 会場までのアクセス

- 大会当日について
  - 当日の受付案内
  - 会議室について
  - フロアガイド
  - 大会スケジュール
  - ホテル及び宿泊について

- その他
  - 大会中のトラブル
  - 基調講演について
  - お問い合わせ先

9月作業開始

・調整が必要な箇所を適宜編集する
